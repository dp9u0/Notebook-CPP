## 返回值

不要返回局部变量的引用或指针

调用返回引用的函数得到左值

返回一个数组：

```cpp
string results[10]{"1", "2", "3", "4", "5", "6", "7", "8", "9", "0"};
string (*example())[10]
{
  return &results;
}
```

还可以使用 using  或者 箭头函数
