# IO

* puts()：只能输出字符串，并且输出结束后会自动换行
* putchar()：只能输出单个字符
* printf() 格式化输出

`%[flags][width][.precision][length]specifier`

* specifier
  * d : 10进制有符号数
  * u : 10进制无符号数
  * o : 8进制无符号数
  * x : 16进制无符号数
  * c : 字符
  * s ：字符串
  * f : 小数形式输出单、双精度
  * e : 指数形式输出单、双精度
  * g : %f或%e中较短的输出宽度输出单、双精度
  * a : 十六进制形式输出浮点数
